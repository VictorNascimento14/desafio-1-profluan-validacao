# Desafio 1 - Validação de Formulário

Este projeto foi desenvolvido como parte de um desafio proposto pelo professor Luan. O objetivo é criar uma página de contato com validação de formulário utilizando HTML, CSS e JavaScript.

## Funcionalidades
- Página de contato com campos para nome, e-mail, telefone e mensagem
- Validação dos campos do formulário
- Feedback visual para erros de preenchimento

## Como executar
1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `contato.html` em seu navegador.

## Estrutura do projeto
- `contato.html`: Página principal do desafio
- `demo/`: Pasta contendo imagens de demonstração
